# 手动创建Git仓库指南

## 方法1: 使用GitHub Desktop (推荐)
1. 下载安装 GitHub Desktop: https://desktop.github.com/
2. 登录GitHub账号
3. 点击 "Add an Existing Repository from your Hard Drive"
4. 选择文件夹: C:\number-drop-game
5. 点击 "Create repository on GitHub.com"
6. 填写仓库信息并发布

## 方法2: 使用VS Code
1. 打开VS Code
2. 打开文件夹: C:\number-drop-game
3. 点击左侧源代码管理图标
4. 点击 "Initialize Repository"
5. 添加所有文件并提交
6. 点击 "Publish to GitHub"

## 方法3: 网页上传
1. 在GitHub创建新仓库: number-drop-game
2. 将所有文件压缩成zip
3. 在GitHub仓库页面上传文件
