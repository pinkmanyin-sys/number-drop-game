# 游戏设计文档

## 核心玩法
- 数字方块从顶部掉落
- 玩家可左右移动方块
- 相同数字碰撞时合并成更大数字（2+2=4, 4+4=8...）
- 目标：创造更大数字，获得高分

## 操作方式
- **移动端**：左右滑动移动，点击旋转
- **PC端**：方向键移动，空格键旋转，下键加速
- **微信小程序**：触摸操作

## 计分规则
- 合并数字获得分数 = 合并后的数字值
- 每1000分升一级
- 速度随等级提升

## 技术特性
- 跨平台兼容
- 云端排行榜
- 实时数据同步
- 响应式UI设计
